# KEP Content Governance Policy

KEP should be useful, but also trusted. This policy protects students from wrong answers, unclear sources, and unsafe data collection.

## 1. Source labels

Every content item must have one of these source labels:

- Official source
- School textbook
- Teacher-created
- Kankor centre paper
- Past paper
- Collected online
- Student submitted
- Other / unsure

If the source is not fully confirmed, do not call it official.

Use labels like:

- Collected practice paper
- Teacher-shared paper
- Centre practice paper
- Unverified past paper

## 2. Review rule

Before publishing, each item should have:

- Source label
- Reviewer name or reviewer code
- Review note
- Status

Suggested statuses:

```text
New
Needs source check
Needs academic review
Needs translation
Approved
Rejected
Published
```

## 3. Question quality checklist

A question is publishable only when:

- The question is clear.
- Four options are provided.
- Only one answer is correct.
- The correct answer is marked.
- Explanation is added where possible.
- Subject and chapter are clear.
- Language is natural and respectful.
- Source is recorded.

## 4. Past paper quality checklist

A paper is publishable only when:

- Year/title is clear.
- Source type is clear.
- PDF/image is readable.
- Permission status is considered.
- It is not falsely labelled as official.

## 5. Privacy rule

KEP should not collect unnecessary personal information from students.

For learning access, avoid requiring:

- Tazkira number
- Gender
- Exact address
- Sensitive personal details

For volunteers, collect only what is needed to contact and organize them.

## 6. Girls Education Space rule

This section should be respectful, accurate, and source-based. Emotional language is allowed, but facts should come from trusted sources such as official education bodies, UN agencies, or recognized human-rights organizations.
