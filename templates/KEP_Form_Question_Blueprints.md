# KEP Google Form Blueprints

Use this if you want to create the forms manually instead of running Apps Script.

## 1. MCQ / Question Submission Form

Fields:

- Contributor name or nickname — short answer — optional
- Contact detail — short answer — optional
- Subject — dropdown — required
- Chapter / topic — short answer — optional
- Language — dropdown — required
- Question text — paragraph — required
- Option A — short answer — required
- Option B — short answer — required
- Option C — short answer — required
- Option D — short answer — required
- Correct answer — multiple choice A/B/C/D — required
- Explanation / reason — paragraph — optional
- Difficulty — dropdown — optional
- Source type — dropdown — required
- Source details / book page / paper year / link — paragraph — optional

## 2. Past Paper Submission Form

Fields:

- Contributor name or nickname
- Contact detail
- Paper title
- Year
- Subject
- Language
- Paper type: Official past paper / Centre practice paper / Teacher shared paper / Collected practice paper / Other
- Source type
- Source details / permission note
- File link
- Notes

## 3. Book / Notes / PDF Submission Form

Fields:

- Contributor name or nickname
- Contact detail
- Resource title
- Subject
- Grade / class
- Language
- Resource type: Textbook / Notes / Formula sheet / Revision guide / Girls self-study pack / Other
- Source type
- Source details / permission note

- Notes

## 4. Correction Report Form

Fields:

- Name or nickname
- Contact detail
- Content type: Question / Answer / Explanation / PDF / Past paper / Translation / Kankor info / Girls Education source / Website bug / Other

- What is wrong?
- Suggested correction
- Language

## 5. Volunteer Application Form

Fields:

- Contact detail

- Languages
