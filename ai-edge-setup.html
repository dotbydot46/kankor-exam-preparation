Setup steps

Deploy in this order

1

Install / open Supabase CLI

Use Supabase CLI for local serving and deployment, or deploy the function from the dashboard if available.

2

Add function secrets

Set OPENAI_API_KEY and OPENAI_MODEL as Supabase function secrets. Do not put them in frontend files.

3

Deploy function

Deploy kep-ai-review from supabase/functions/kep-ai-review/index.ts.

4

Login as admin/reviewer

The function checks your Supabase login and profile role before returning suggestions.

5

Open AI Assistant

Fill a question and click Run Backend AI Review.

6

Reviewer approves manually

AI suggestions are never published automatically.
