Troubleshooting

Common problems and fixes

“Connection failed” on Database Setup

Check the project URL and public key. Make sure you did not paste a secret/service-role key. Make sure the main v4.0 SQL was run.

“Logged in but role is student” on Review DB

Open Supabase Table Editor → profiles → your row → set role to admin or reviewer.

Practice DB loads zero questions

Add a question and mark its status as published, or run the sample content SQL.

Library DB loads zero resources

Add resources to the resources table with status published, or run the sample content SQL.

Study plan does not save to database

Run scripts/KEP_v4_4_Study_Plans_Table.sql. The page still saves a local fallback even without the table.

Old header or logo appears

Hard refresh the page or clear old service worker/browser cache. v5.1 uses the consistent header/logo system.
