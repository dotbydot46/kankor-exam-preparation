# KEP v3.9 — Mock Exam Guide

This version upgrades the Mock Exam page into a timed exam engine using reviewed Published questions from Google Sheets.

## What it does

- Loads only `Published` questions from your KEP Google Sheet.
- Supports Quick Mock, Subject Mock, and flexible Full Kankor-style Mock.
- Includes timer, question navigation, score, percentage, time used, subject breakdown, and wrong-answer review.
- Saves the mock result locally so the Student Dashboard can use it.
- Shows a safe field-readiness estimate.

## Important honesty rule

KEP must not promise official Kankor admission, field placement, or university selection.

The result page uses:
- Strong / Medium / Needs work
- Readiness estimate
- This is a study estimate only

Official placement depends on official exam results, yearly capacity, competition, and NEXA rules.

## Real Kankor rules

The Full Kankor-style template is flexible for now.

Before calling it the real official format, verify:
- total questions
- exam duration
- subject distribution
- questions per subject
- marking rules
- yearly changes
- source link/date checked
